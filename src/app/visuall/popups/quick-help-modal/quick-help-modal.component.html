Quick Help
File > Import GFA: To import a GFA file to the database
Data > Clear Data: To clear the database
Map > Query by Rule: To query the database with segments/links of interest
Click: To inspect object properties
Click segment: To enable visual cues for expanding the view with up/downstream
Right-click: Context menu for associated object
Shift + drag: Box selection (select touched nodes and included edges)
Shift + click: Add/remove an element to/from the current selection
Tips
Settings > Emphasize on hover: To focus on a node (edge) and its neighbors (source/target) on mouse over
Select multiple objects: To see their attributes as a table
