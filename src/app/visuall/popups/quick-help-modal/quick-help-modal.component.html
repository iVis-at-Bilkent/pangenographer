Quick Help
File > Import GFA: To import a GFA file into Freebase.
File > Samples > Freebase: To switch to Freebase.
File > Samples > Minigraph GRCh38 Human Pangenome: To load a variation graph built by Minigraph-r518 on the GRCh38 reference, representing HPRC year-1 human samples, excluding HG002 and NA19240 (source).
File > Samples > Human HLA Pangenome: To load a variation subgraph extracted from the GRCh38 pangenome (Minigraph GRCh38 Human Pangenome) using the CRHPG script (Andreace et al., Genome Biology, 2023) to model diversity across the HLA-A region.
File > Samples > Bifrost E.coli Pangenome: To load a colored de Bruijn graph constructed by Bifrost from 100 E.coli assemblies (k=31, 10 threads) to capture strain-level variation.
File > Samples > PGGB E.coli Pangenome: To load a PGGB-built variation graph of 100 E.coli strains, smoothed and fixed to comprehensively represent genomic variation.
Data > Clear Data: To clear the database.
Map > Query by Rule: To query the database using segments or links of interest.
Database > Custom Queries: To construct an initial graph of interest.
Click: To inspect object properties.
Click segment: To enable visual cues for expanding the view upstream and downstream.
Right-click: To open the context menu for the associated object.
Shift + drag: To select as a box (select touched nodes and included edges).
Shift + click: To add or remove an element to or from the current selection.
Tips
Settings > Emphasize on hover: To focus on a node or edge and its neighbors (source or target) on mouse over.
Select multiple objects: To see their attributes as a table.
